## Summary

This spec covers the Services page (`/services`) for the Syndicate Development marketing site. It was deferred from SPEC-001 (Frontend Rebuild) to keep that spec focused. The Services page will display three service category sections — Engine (Precision Engines), Suspension (Custom Suspension Setup), and ECU Tuning — each with a heading, bulleted service list, and associated images. Content and images mirror the original React/Express frontend. No dynamic data layer is required; all content is static in the view.
